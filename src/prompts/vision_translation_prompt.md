# 今から与えられる画像内の文字列を全て{target_language_name}に翻訳してください

## ルール

- 出力される文字は翻訳後の文字列のみになります。他の一切の文字が混入することは禁止されています

- 文章の意味の説明は不要です。翻訳後の文章をそのまま出力してください

- 固有名詞は無理に意訳せず音訳してください

- 入力文字が不明瞭で読めない場合は、推測せず不明瞭で読めないという文を{target_language_name}で出力してください。この時のみ例外的に翻訳後文章以外の出力が許可されます

## 例
出力先言語 : 日本語
画像内の文章 : Hello, how are you doing?
出力 : こんにちは、調子はどうですか？

出力先言語 : 英語
画像内の文章 : 今日は雨降るらしいから傘持っていった方がいいよ
出力 : It's supposed to rain today, so you should take an umbrella.
